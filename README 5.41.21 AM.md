# Valentine Proposal Site

A lightweight single-page site you can host on GitHub Pages.

## Customize
- Replace `assets/icon.svg` with your custom icon (keep the filename or update `index.html`).
- Edit the message text in `index.html`.

## Deploy on GitHub Pages
1. Create a new GitHub repo and push this folder.
2. In GitHub, go to Settings -> Pages.
3. Set Source to `Deploy from a branch`, choose `main` and `/ (root)`.
4. Save. Your site will be available at the URL GitHub provides.
